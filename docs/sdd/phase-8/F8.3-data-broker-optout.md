# F8.3 Data Broker Opt-Out - SDD

## Overview
Guided data broker opt-out tool with a database of 10 major brokers, step-by-step instructions, difficulty ratings, category filtering, and persistent opt-out progress tracking.

## Data Layer

### `apps/web/src/data/data-brokers.ts`
- Database of 10 data brokers: Spokeo, Whitepages, BeenVerified, Intelius, PeopleFinders, Acxiom, LexisNexis, Epsilon, Oracle Data Cloud, TruthFinder
- Each broker includes: name, category, website, opt-out URL, method, difficulty, estimated time, data types, step-by-step instructions
- Categories: people-search, marketing, financial, background-check
- Difficulty levels: easy (5 min), medium (10-15 min), hard (15-20 min)
- Helper functions for filtering by category and listing categories

## Components Created

### `apps/web/src/components/data-brokers/BrokerCard.tsx`
- Expandable card with broker name, category, difficulty badge, and data type chips
- Expanded view shows estimated time, opt-out method, numbered instructions
- "Open Opt-Out Page" link opens external URL in new tab
- "Mark Done" button records completion in store

### `apps/web/src/components/data-brokers/OptOutProgress.tsx`
- Overall progress bar with percentage and exposure level (Low/Moderate/High)
- Completed vs remaining counts
- Difficulty breakdown showing easy/medium/hard counts

### `apps/web/src/components/data-brokers/index.ts`
- Barrel export for BrokerCard and OptOutProgress

## Page

### `apps/web/src/pages/DataBrokerOptOutPage.tsx`

- Category filter buttons with counts
- Sort by difficulty, name, or category
- Green styling for already-opted-out brokers
