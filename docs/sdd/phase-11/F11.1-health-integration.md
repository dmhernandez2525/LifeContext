# F11.1 Health Integration - SDD

## Overview
Added health metrics tracking with 6 metric types (steps, sleep, heart rate, weight, water, exercise), daily summaries, goal progress, and a weekly steps chart.

## Files Created

### `apps/web/src/lib/healthMetrics.ts`
- 6 metric types: steps, sleep, heartRate, weight, water, exercise
- `METRIC_CONFIG` with labels, units, icons, colors, and default goals
- `buildDailySummary()` - Aggregates entries by type for a date
- `getWeeklyAverage()` - Calculates 7-day average for a metric
- `getGoalProgress()` - Returns percentage toward goal
- `formatMetricValue()` - Human-readable value formatting
- `getLast7Days()` - Returns last 7 date strings

### `apps/web/src/store/health-store.ts`
- Zustand persisted store with `entries: HealthEntry[]` and `goals` map
- Actions: addEntry, removeEntry, importEntries (with dedup), setGoal, clearAll
- Import deduplication via date+type+value+source composite key

### `apps/web/src/components/health/HealthDashboard.tsx`
- 6-card grid showing each metric with value, goal progress bar, and percentage
- Mini bar chart showing steps for the last 7 days
- Current day highlighted in chart

### `apps/web/src/components/health/HealthEntryForm.tsx`
- Quick-add form with metric type selector, value input, and add button
- Dynamic unit label and placeholder based on selected metric
- Step size adapts to metric type (0.1 for sleep/weight, 1 for others)

### `apps/web/src/pages/HealthPage.tsx`
- Date navigator with previous/next day controls
- Quick-add form panel and HealthDashboard
- Today/Yesterday labels, disabled forward navigation past today

## Files Modified

### `apps/web/src/App.tsx`
- Added HealthPage import and `/app/health` route
