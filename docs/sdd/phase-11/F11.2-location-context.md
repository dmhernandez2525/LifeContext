# F11.2 Location Context - SDD

## Overview
Added location-aware context with Geolocation API integration, saved places with proximity detection, visit tracking, and a location history dashboard.

## Files Created

### `apps/web/src/lib/locationContext.ts`
- `haversineDistance()` - Distance between two coordinates in km
- `isNearPlace()` - Check if coordinates are within a place's radius
- `detectPlace()` - Find nearest saved place within radius
- `formatCoords()` - Human-readable coordinate formatting
- `formatDistance()` - Distance display (m or km)
- `groupByDate()` - Group location entries by date
- `getVisitSummary()` - Aggregate visit counts per place for last N days
- `DEFAULT_PLACE_ICONS` - 10 emoji options for saved places

### `apps/web/src/store/location-store.ts`
- Zustand persisted store for location entries, saved places, and tracking state
- Actions: addEntry, removeEntry, addPlace, updatePlace, removePlace, setTracking, setLastLocation, incrementVisit

### `apps/web/src/components/location/LocationTracker.tsx`
- Current location display with tracking on/off toggle
- Browser Geolocation API integration with error handling
- Detected place name display when near a saved place
- Manual location refresh button

### `apps/web/src/components/location/PlaceManager.tsx`
- Add new saved places from current location
- Icon selector with 10 emoji options
- Configurable detection radius (100m, 200m, 500m, 1km)
- Place list with visit count and last visit date
- Delete places

### `apps/web/src/pages/LocationPage.tsx`
- Stats bar: location points, days tracked, saved places
- LocationTracker widget
- Recent visits summary (last 30 days)
- PlaceManager for saved places
- Privacy info banner

## Files Modified

### `apps/web/src/App.tsx`
- Added LocationPage import and `/app/location` route
