# F10.2 Day One Import - SDD

## Overview
Added Day One journal import with JSON parser, entry preview with selection, and IndexedDB storage integration.

## Files Created

### `apps/web/src/lib/importers/types.ts`
- Shared import types: `ParsedEntry`, `ImportResult`, `ImportError`, `ImportSource`
- `ParsedPhoto` interface for photo metadata
- `SOURCE_LABELS` map for display names

### `apps/web/src/lib/importers/dayone.ts`
- `parseDayOneExport(json)` - Parses Day One JSON format, returns entries and skip count
- `isDayOneFormat(json)` - Validates file structure
- `inferMood(text)` - Simple sentiment analysis mapping to JournalMood
- Location and weather formatting helpers
- Photo metadata mapping

### `apps/web/src/lib/importers/importHandler.ts`
- `importEntries(entries)` - Saves parsed entries to IndexedDB journal table
- Deduplication via `import:<sourceId>` tags
- Appends location/weather metadata to content
- Returns ImportResult with imported/skipped/error counts

### `apps/web/src/components/import/ImportPreview.tsx`
- Scrollable entry list with checkbox selection
- Select all / deselect all toggle
- Date range summary, tag display, location/star indicators
- Confirm/cancel actions with importing state

### `apps/web/src/components/import/DayOneImport.tsx`
- Four-phase flow: upload, preview, complete, error
- Drag-and-drop file zone with click-to-browse fallback
- File validation before parsing
- Success summary with import counts

### `apps/web/src/pages/ImportPage.tsx`
- Unified import hub with tabbed source selector
- Day One active, Journey and Apple Notes marked "Coming Soon"
- Info banner about local storage and encryption

## Files Modified

### `apps/web/src/App.tsx`
- Added ImportPage import and `/app/import` route

## Architecture Notes
- Import tags (`import:<sourceId>`, `source:<source>`) enable deduplication and filtering
- Content stored as unencrypted EncryptedData placeholder (version: 0) for imported entries
- Shared types in `importers/types.ts` reused by all three import parsers (F10.2-F10.4)
