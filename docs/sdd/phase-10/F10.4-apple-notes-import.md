# F10.4 Apple Notes Import - SDD

## Overview
Added Apple Notes import supporting HTML and plain text file exports with HTML-to-text conversion, date extraction from meta tags and filenames, and multi-file upload.

## Files Created

### `apps/web/src/lib/importers/apple-notes.ts`
- `parseAppleNotesExport(files)` - Parses array of { name, content } file objects (HTML or TXT)
- `isAppleNotesFormat(files)` - Validates file extensions
- `htmlToText(html)` - Strips HTML tags, decodes entities, preserves paragraph breaks
- `extractTitle(html)` - Pulls title from h1, title tag, or first line
- `extractDate(html, filename)` - Checks meta tags, then filename date patterns, falls back to now
- `extractFolder(html)` - Reads folder from meta tag if present

### `apps/web/src/components/import/AppleNotesImport.tsx`
- Multi-file upload with drag-and-drop (accepts .html, .htm, .txt)
- Four-phase flow matching Day One and Journey importers
- File count and type validation before parsing
- Reuses shared ImportPreview component

## Architecture Notes
- Unlike Day One and Journey (single JSON file), Apple Notes uses multi-file upload (one HTML/TXT per note)
- HTML-to-text conversion preserves paragraph structure while stripping all markup
- Date extraction falls back gracefully: meta tags, filename patterns (YYYY-MM-DD), then current date
- Folder metadata preserved as `folder:` tag for filtering imported notes by original Notes folder
