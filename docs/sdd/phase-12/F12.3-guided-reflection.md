# F12.3 Guided Reflection - SDD

## Overview
Added guided reflection with 6 categories of prompts, interactive session flow with progress tracking, and category selection UI.

## Files Created

### `apps/web/src/lib/guidedReflection.ts`
- 6 categories: gratitude, growth, relationships, health, goals, emotions
- 5 prompts per category (30 total), randomly selected per session
- `generateReflectionSession()` - Creates session with prompts from selected or random categories
- `isSessionComplete()` - Checks all prompts have responses
- `getSessionProgress()` - Returns completion percentage
- `CATEGORY_CONFIG` with labels, icons, and colors

### `apps/web/src/components/ai/ReflectionSession.tsx`
- Progress bar with question counter and completion percentage
- Category badge with icon for current prompt
- Auto-resizing textarea for responses
- Dot navigation showing answered/unanswered/current prompts
- Next button progresses through prompts
- Complete button (enabled when all answered)
- Completed state with restart option

### `apps/web/src/pages/ReflectionPage.tsx`
- Category grid for topic selection
- Start button (with selected count or "Random")
- Session view replacing category selector during active session
- Restart returns to category selection

## Files Modified

### `apps/web/src/components/ai/index.ts`
- Added ReflectionSessionView export

### `apps/web/src/App.tsx`
- Added ReflectionPage import and `/app/reflection` route
